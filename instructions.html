Page Proofer - Version 1.0

Description

Easily embed Page Proofer code into your Business Catalyst site. What makes this App unique from just adding the default Page Proofer code snippet is that it automatically enables pageproofer if the user is logged into the backend or logged in on the Business Catalyst Domain. This makes it easy for clients to give feedback durring development and after the page goes live.

Developer Notes

Tag to Enable App on Website

Place in the head of the template

{% include "/_System/apps/solid-page-proofer/_config/pageProoferCode.inc" -%}

Finding Page Proofer Account ID and Site ID

Log into Page Proofer click on the site you are installing it on. See the following address URL.
https://app.pageproofer.com/[Account ID]/dashboard/[Site ID]/

Remove Page Proofer

WARNING: Removing this application will delete the app. No modifications will be made to your pages outside the app folder.
