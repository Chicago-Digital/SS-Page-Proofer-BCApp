Description

Easily embed Page Proofer code into your Business Catalyst site. What makes this App unique from just adding the default Page Proofer code snippet is that it automatically enables pageproofer if the user is logged into the backend or logged in on the Business Catalyst Domain.

PageProofer now allows for all users to submit feedback, not just users that have a PageProofer account (not available on all PageProofer Plans). When allowing all users to submit feedback, this app passes in the users email address they use to login to Business Catalyst, saving the user time.

Developer Notes

Tag to Enable App on Website

Place before the closing body tag of the template

{% include "/_System/apps/solid-page-proofer/_config/pageProoferCode.inc" -%}

Finding Page Proofer Account ID and Site ID

Log into Page Proofer click on the site you are installing it on. See the following address URL.
https://app.pageproofer.com/[Account ID]/dashboard/[Site ID]/

New in Version {{appManifest.version}}

  • Supports PageProofer's new anonymous feature, allowing users without a pageproofer account to give feedback
  • Only shows PageProofer's feedback button on the live website when a user has logged into the Business Catalyst admin
  • When users submit using the feedback button, this app auto passes the users same email address they use to login to Business Catalyst, making giving feedback even easier
